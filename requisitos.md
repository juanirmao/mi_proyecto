ROI' S ( REQUERIMIENTO ORDINARIO DE INFORMACIÓN) 

1. De las diferentes plantillas de los ROIS se requiere que automaticamente según sea el caso, el sistema indique si es un posible agentes especial o no. Ej. ROI BASICO de 5 puntos. El posible agente especial da respuesta a cada punto y el sistema determina si es un agentes especial y se espera que genenere un resultado
NOTA: El ROI determina quien es el posible Sujeto pasivo ( el posible agente especial) 
Ej: Banco Bogotá ( sujeto Pasivo, posible agente especial), Sujeto Activo : Municipio La Calera 
2. Genere alerta o mensaje 

Ejemplo: 1. CELSIA ( comercializadora, generadora, transimora de Energia ) cobra el IAP impuesto de alumbrado Público. 
 2. CELSIA es un posible sujeto pasivo. como determinamos si es un posible sujeto pasivo: ROI empresas de Energía.

INVITACIÓN A PAGO 


1 se requiere que se sistematice  la proyección de la liquidación del IAP, según cada acuerdo Municipal 
2. Se requiere la sistematizacion de la proyeccion del cobro del IAP de los sujetos pasivos identificados.
3. Se requeire la sistematizacion y enlace de los informes de facturacion y recaudo - Cartera  de las categorias especiales 
4. Se requeire  el cruce de la base de datos suministrada por los diferentes municipios, y el excel del acuerdo municipal para ubicarlos en las diferentes categorias. 
4. una vez este este cruce, se enlace directamente con el ROI que le corresponda según sea el caso para proceder al envio al municipio. 
5. A su vez este ROI este enlazado según corresponda la respuesta lo envie a la invitación a pago
6. La invitación a pago este enlazada con la Liquiedación del IAP.
